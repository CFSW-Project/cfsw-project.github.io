1. CSS background-color: This property sets the background color of an element. You can use color names, HEX codes, RGB, or other color formats. .example { background-color: lightblue; }

CSS margin: This property sets the space outside the border of an element. It can take one, two, three, or four values to specify margins for all sides or individual sides. .example { margin: 20px; /* All sides will have 20px margin */ } .example { margin: 10px 15px; /* Top and bottom margins are 10px, left and right margins are 15px */ }

CSS padding: This property sets the space inside the border of an element, between the border and the content. It can also take one, two, three, or four values. .example { padding: 15px; /* All sides will have 15px padding */ } .example { padding: 10px 20px; /* Top and bottom padding are 10px, left and right padding are 20px */ }

CSS height: This property sets the height of an element. It can be specified in various units like pixels, percentages, ems, etc. .example { height: 200px; /* Sets the height of the element to 200 pixels */ }

CSS width: This property sets the width of an element. It can also be specified in pixels, percentages, ems, etc. .example { width: 50%; /* Sets the width of the element to 50% of its parent container */ }

It can be used to create circular or elliptical corners. .example { border-radius: 10px; /* Rounds all corners with a radius of 10 pixels */ } .example { border-radius: 50%; /* Creates a circle if the width and height are equal */ }

CSS margin-left: This property moves an element to the left by a specified amount using pixels or other units.

CSS margin-right: This property moves an element to the right by a specified amount using pixels or other units.
